# EndField

### 基礎數值

- 敏捷：物理抗性：物理傷害減免x%
- 力量：生命值
- 智識：法術抗性：法術傷害減免x%
- 意志：受治療效率

---

- 主副能力：攻擊力

### 傷害類型

- 物理異常
    - 猛擊
        - 敵人未破防
            - `破防`＋1
        - 敵人已破防
            - `破防` = 0, 大量物理傷害
    - 碎甲
        - 敵人未破防
            - `破防`＋1
        - 敵人已破防
            - `破防` = 0, 物理傷害, 施加`物理易傷` （碎甲狀態）
    - 倒地
        - 敵人未破防
            - `破防`＋1
        - 敵人已破防
            - `破防` +1, 物理傷害, `失衡值`++, 敵人跌倒不攻擊（倒地狀態）
    - 擊飛
        - 敵人未破防
            - `破防`＋1
        - 敵人已破防
            - `破防` +1, 物理傷害, `失衡值`++, 敵人懸浮不攻擊(擊飛狀態）
- 法術附著（最多四層）
    - `灼熱附著`
        - 灼熱爆發（法術爆發）：灼熱附著2+
        - 燃燒 🔥（法術異常）：其他法术附着（电/冰/自然） + `灼热附着`。敵人持續受到灼熱傷害
    - `電磁附著`
        - 電磁爆發（法術爆發）：電磁附著2+
        - 導電 ⚡（法術異常）: 其他法术附着（火/冰/自然） + `電磁附著`。敵人受到的**法術傷害**提高
    - `自然附著`
        - 自然爆發（法術爆發）：自然附著2+
        - 腐蝕 🤢 (法術異常）: 其他法术附着（火/冰/電） + `自然附著`。敵人所有抗性逐漸下降。
    - `寒冷附著`
        - 寒冷爆發（法術爆發）：寒冷附著2+
        - 凍結 ❄️ (法術異常）: 其他法术附着（火/自然/電） + `寒冷附著`。敵人無法行動
            - 若受到`破防` or `物理異常`：碎冰 🧊

✏️ 破防與附著均為”狀態”, 本身不具有任何效果。最多都累積 4 層。

### Buff 類型

- 脆弱：目標受到的類型傷害++, 可疊加, 作用時間比碎甲長～
- 虛弱：一定時間內目標造成傷害降低
- 連擊：全隊的下個戰技 or 終結技傷害提高 30% / 20%.
- 增福：一定時間內造成的傷害提高
- 庇護：一定時間內受到傷害降低
- 護盾：吸收傷害
- 源石技藝強度
    - **猛擊, 燃燒** 🔥 **, 凍結** ❄️ **, 碎冰** 🧊 ****: 傷害+
    - **碎甲, 導電** ⚡ **, 腐蝕** 🤢               ：傷害++
    - **擊飛, 倒地**                                     ：傷害+, `失衡值`+

### 其他名詞

- 失衡節點：部分敵人存在, 當失衡值累積到失衡節點時, 打斷敵人行動並讓敵人踉蹌
